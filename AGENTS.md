# Repo Agent Instructions

This repository follows the global agent policy supplied by the active local agent environment.

## Autonomous Learning Loop

For substantive tasks:
1. Keep `AGENTS.md`, `docs/notes/index.md`, and `docs/notes/_template-learning-note.md` present.
2. Run a lesson check at task end.
3. If reusable, add/update a note in `docs/notes/` and update `docs/notes/index.md` in the same change.
4. If not reusable, include `no reusable lesson` in the final summary.

## Lesson Check
- Did I hit an error pattern?
- Did I fix a non-obvious bug?
- Did I choose one architecture tradeoff over another?
- Did performance behavior change due to a specific fix?
